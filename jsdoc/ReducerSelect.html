ReducerSelect

Mirrors a simplified behaviour of the switch statement in javascript. It creates a reducer which accepts the form of:

select({
  in: [
    { case: Reducer,  do: Reducer },
    { case: Reducer,  do: Reducer }
    ...
  ],
  default: Reducer
})

case blocks are executed sequentially, if a case returns a truthy value its do statement will be resolved and returned.

If no case matches then the default reducer will be resolved.
